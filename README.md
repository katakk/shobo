# shobo

foltiaの録画ファイル名をタイトル名に変えて参照リンク精製するやつ

cron で定間隔で回すとsamba みやすくなる

    \\192.168.x.x\mp4\アニメ\2017_1Q\うらら迷路帖\うらら迷路帖 #1 20170106-2330 26ch.MP4

みたいな感じであくせすできるようになる
  
  
## samba

    [record]
    path = /record
  
    [record2]
    path = /record2
    
    [mp4]
    path = /mp4
    
    [raw]
    path = /raw
    
    [anime]
    path = /anime
